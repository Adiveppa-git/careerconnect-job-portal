# CareerConnect – Full Stack Job Portal

CareerConnect is a production-ready job portal built using React, FastAPI, and MongoDB Atlas.

## Tech Stack

- React.js
- Tailwind CSS
- FastAPI
- MongoDB Atlas
- JWT Authentication
- Git & GitHub
- Vercel
- Render

## Features

- Job Seeker Dashboard
- Recruiter Dashboard
- Admin Dashboard
- JWT Authentication
- Resume Upload
- Job Search & Filters
- Application Tracking
- Notifications
